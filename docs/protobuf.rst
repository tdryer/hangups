Protocol Buffers
================

This file documents the reverse-engineered `Protocol Buffers`_ used by the
Hangouts chat protocol. hangups and other projects may these Protocol Buffers
to serialize and deserialize data for communicating with Google's servers.

Some of the most important Protocol Buffer messages include:

- `Conversation`_
- `Entity`_
- `Event`_
- `StateUpdate`_

.. _`Protocol Buffers`: https://developers.google.com/protocol-buffers/
